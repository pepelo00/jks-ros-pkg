Developer Notes
---------------


REQUIREMENTS


The following software packages are required 
to generate the distribution archive:

* Microsoft Visual Studio 2008

* Slik  SVN (http://www.sliksvn.com/en/download/)
    OR
  cygwin 'subversion' package (and cygwin/bin in the system PATH)
    OR
  TortoiseSVN with command line client installed

* doxygen (http://www.stack.nl/~dimitri/doxygen/)

* graphviz (http://www.graphviz.org/Download..php)


To produce the installers, run the distrib.bat script.
